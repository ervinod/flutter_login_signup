# Flutter Login Signup UI Demo


This project demostrates use of TMDB movie API
.

Following components are used in this Flutter project:

- [Splash Screen]
- [Login UI]
- [Signup UI]
- [Forgot Password UI]
- [Login Signup Validation]
- [Custom App Bar]
- [Bottom App Bar]
- [Custom Alert]
